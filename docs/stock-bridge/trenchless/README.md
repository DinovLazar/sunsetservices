# Trenchless — stock-bridge images

Temporary bridge images for the Trenchless & Directional Boring service pages (Brand Guide §9.1/§9.3). **Replace by 2026-10-01** with authentic Sunset photography.

Drop downloaded JPGs here using the exact filenames from `../stock-image-manifest.md`.

**Filename convention (mandatory):** `stock-trenchless-<service-slug>-<hero|support>-<NN>.jpg`
Example: `stock-trenchless-trenching-excavation-hero-01.jpg`

The `stock-` prefix is required — it keeps bridge images permanently distinguishable from real Sunset project photos. After saving each file, fill its four download-time cells (source URL, photographer, license, download date) in the manifest.

**Accuracy is stricter here than for Waterproofing.** The audience is commercial/municipal — the equipment shown must genuinely be the equipment for that service (a directional-drill rig is not a mini-excavator). When no accurate photo exists after a real search of all three sources, **stop and flag the service as "diagram needed"** in the manifest — do not settle for a close-enough photo.

**Missile Boring** and **HDPE Pipe Fusing** take no photo (diagram needed). **Handhole / Pull Box** is source-if-truthful, else diagram. See the manifest.

Sources: Unsplash / Pexels / Pixabay only. No AI-generated images. Hero ≥1600 px wide (landscape, prefer 2000+); support ≥1200 px.
