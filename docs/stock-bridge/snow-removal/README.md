# Snow Removal — stock-bridge images

Temporary bridge images for the Snow Removal service pages and the `/snow-removal/` division landing hero (Brand Guide §9.1/§9.3). **Replace by 2027-01-31** with authentic Sunset photography — real snow photos cannot exist until the first 2026–27 snowfall, so this bridge intentionally carries a later expiry than the 2026-10-01 waterproofing/trenchless images.

Drop downloaded JPGs here using the exact filenames from `../stock-image-manifest.md`.

**Filename convention (mandatory):** `stock-snow-removal-<service-slug>-<hero|support>-<NN>.jpg`
Example: `stock-snow-removal-de-icing-hero-01.jpg`

The `stock-` prefix is required — it keeps bridge images permanently distinguishable from real Sunset project photos. After saving each file, fill its four download-time cells (source URL, photographer, license, download date) in the manifest.

**Accuracy is binding — an inaccurate photo is worse than none.** The photo must show the actual subject of the page (de-icing salt, shoveling, snow blower, plow truck). When no accurate photo exists after a real search of all three sources, **stop and record a GAP** in the manifest — do not settle for a close-enough photo.

Sources: Unsplash / Pexels / Pixabay only. No AI-generated images. Hero ≥1600 px wide (landscape, prefer 2400+ — the hero derivative is 2400 px wide and must not upscale).
