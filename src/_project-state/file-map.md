# File Map

> One line per non-trivial file or directory. Keep it skim-able. Updated end of every phase.

## Project root

- `Sunset-Services-Plan.md` — master site plan and spec (canonical).
- `docs/design-handovers/Part-1-Phase-{03,05,06,08,11,13}-Design-Handover.md` — Claude Design handovers, one per design phase. Phase 1.11 covers About + Contact (input for Phase 1.12); Phase 1.13 covers Service Areas index + 6 location pages (input for Phase 1.14).
